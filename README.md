# ASTROS
A code to convert a Robocore's Rocket Tank chassis into a Pioneer P3-DX like robot.

# Used materials
* An ESP-32 module;
* 2 Motors with rotary encoders;
* Robocore's Rocket Tank chassi;
* H-bridge module;
* PlatformIO with Arduino Framework.
